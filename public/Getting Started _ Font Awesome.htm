Manage Font Awesome as a Package

Use npm in your projects? We have an official package you can install and update from more easily than downloading.

/path/to/your-awesome-project

npm install @fortawesome/fontawesome-free 
npm install @fortawesome/fontawesome-free

Looking for more advanced options? We have Angular, Ember, React, Vue specific packages as well as an API.

There are a few steps involved after you install. We'll help you understand what to do with the files to get up and running.
